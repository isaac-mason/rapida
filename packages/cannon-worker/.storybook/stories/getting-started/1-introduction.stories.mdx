# `@rapidajs/cannon-worker`

`cannon-worker` is library that can be used with `three` to bring physics into your world!

## Installation

To get started with using `cannon-worker` alongside three js, install `cannon-worker` and `three`

```bash
# Install rapida-physics and three
> npm install @rapidajs/rapida-physics three

# Optionally, also install three-stlib if you want to use it
> npm install three-stdlib
```
