# `@rapidajs/cannon-worker`

`@rapidajs/cannon-worker` is library that can be used with `three` to bring physics into your world!

## Installation

To get started with using `@rapidajs/cannon-worker` alongside three js, install `@rapidajs/cannon-worker` and `three`

```bash
# Install cannon-worker and three
> npm install @rapidajs/cannon-worker three

# Optionally, also install three-stlib if you want to use it
> npm install three-stdlib
```
