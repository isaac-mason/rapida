# `@rapidajs/three`

`@rapidajs/three` is a library that helps you create 3D content with three.js.

**Highlights**
- [x] Build your 3D scenes with three.js
- [x] Framework agnostic
- [x] Support for multiple cameras, scenes and views out of the box!
- [x] Support for AR and VR with WebXR

## Installation

To get started, install `@rapidajs/three` and it's peer dependencies

```bash
npm install @rapidajs/three three
```
