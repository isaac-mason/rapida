# `@rapidajs/rapida-physics`

`rapida-physics` is library that can be used with `three` to bring physics into your world!

## Installation

To get started with using `rapida-physics` alongside three js, install `rapida-physics` and `three`

```bash
# Install rapida-physics and three
> npm install @rapidajs/rapida-physics three

# Optionally, also install three-stlib if you want to use it
> npm install three-stdlib
```
